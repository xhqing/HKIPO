# 港股打新知识笔记

> 记录港股IPO相关的基础概念、规则术语和关键知识点。

## 一、IPO基础架构

### 1.1 公开发售 vs 国际配售

| 概念 | 说明 |
|------|------|
| 公开发售（Public Offering） | 面向散户投资者的部分，个人投资者申购的池子 |
| 国际配售（International Placement / 国配） | 面向机构投资者（基金、券商等）的部分，份额通常占大头 |

香港IPO将总发行量分为两部分，分别面向散户和机构。散户能买多少，取决于公开发售的比例。

### 1.2 回拨机制（Clawback Mechanism）

> 详见下方"回拨机制"章节。

## 二、回拨机制（Clawback）

### 2.1 基本概念

回拨是指当散户认购热度较高时，从国际配售池子中"拨"一部分股票到公开发售，让散户能买到更多。

### 2.2 回拨类型

| 类型 | 说明 |
|------|------|
| 正常回拨 | 公开认购倍数达到一定门槛后，按规则触发回拨 |
| 套路回拨（惜货回拨） | 承销商主动压低国配认购量，限制回拨比例（如从50%限至20%），筹码集中在机构手中，利好股价但中签极难 |
| 主动回拨 | 公开认购不足15倍时，承销商强行将国配回拨至公开发售 → **大概率破发（一票否决项）** |

### 2.3 回拨比例与破发率关系

| 回拨比例 | 破发率 | 解读 |
|---------|-------|------|
| 未触发回拨 | ~36% | 散户筹码少，抛压轻 |
| 回拨 30% | ~46% | 散户获配增多 |
| 回拨 40% | ~59% | 散户砸盘重灾区 |
| 回拨 50% | ~30% | 极端热门，机构也抢 |

**核心规律**：回拨40%时破发率最高——散户拿太多筹码，首日集中抛售压力最大。

### 2.4 2025年新规：机制A vs 机制B

| 规则 | 机制A | 机制B |
|------|-------|-------|
| 采用比例 | ~13% | **~87%（主流）** |
| 公开发售比例 | 5%-35%（按认购倍数递增） | **固定（通常10%）** |
| 回拨 | 有（按倍数触发） | **无回拨** |
| 中签率 | 较高 | **极低（中位数1%-2%）** |

**机制B的核心影响**：无论散户多火爆，公开发售永远只占10%，中签率断崖式下跌。

## 三、基石投资者（Cornerstone Investors）

### 3.1 定义

在IPO正式招股前就提前签订协议、承诺认购一定数量股票的机构投资者。

### 3.2 核心特征

| 特征 | 说明 |
|------|------|
| 提前锁定 | 招股前已签约认购 |
| 锁定期 | 通常承诺 **6个月不卖出** |
| 定价 | 无折扣，按发行价买入 |
| 公开披露 | 名字和认购金额会列在招股书中 |

### 3.3 为什么重要

基石投资者用真金白银和锁仓半年为新股"背书"，是判断新股质量的重要信号。

### 3.4 判断标准

| 标准 | 评级 | 说明 |
|------|------|------|
| 知名机构≥3家，占比25%-40% | ⭐⭐⭐ 强信号 | 黄金区间：太多减少流通盘，太少缺乏信心 |
| 有基石，占比10%-25% | ⭐⭐ 中性 | |
| 基石占比<10% | ⭐ 弱信号 | |
| 无基石或"假基石" | ❌ 负面 | 假基石=关联方为保发行，见下方 |

### 3.5 假基石（一票否决项）

基石投资者实际上是公司关联方或"关系户"，目的仅为保证发行成功，无实质背书意义。

## 四、甲乙组（Tranche A / Tranche B）

### 4.1 分组规则

| 组别 | 认购金额 | 特点 |
|------|---------|------|
| 甲组（Tranche A） | < 500万港元 | 散户组，资金门槛低 |
| 乙组（Tranche B） | ≥ 500万港元 | 大户组，需大资金 |

### 4.2 关键术语

| 术语 | 说明 |
|------|------|
| 甲头 | 甲组中**金额最小**的档位（如一手党），是甲组的"开头" |
| 甲尾 | 甲组中**金额最大**的档位（~500万），紧贴乙组门槛，是甲组的"末尾" |
| 乙头 | 乙组中认购金额最小的，刚过500万门槛 |
| 顶头槌（Top Spinning） | 乙组中认购金额最高的，通常为公众可申购的上限 |

**分组示意图**：

```
甲头（一手） ──→ ………… ──→ 甲尾（~500万）│ 乙头（~500万）──→ ………─→ 顶头槌
└──────── 甲组 ≤500万 ────────┘└──────── 乙组 >500万 ──────────┘
```

> "头" = 各组起点（金额小），"尾" = 各组终点（金额大）。甲尾和乙头虽然都贴近500万门槛，但分属不同组别，适用不同的中签率分配规则。

### 4.3 新规后：甲尾反超乙头

机制B下，投资者扎堆乙组导致过度拥挤，**甲尾中签率可能反超乙头**：

- 双登股份：甲尾35% vs 乙头13%
- 佳鑫国际资源：甲尾70% vs 乙头63%

**策略启示**：机制B新股优先选择甲尾认购，节省资金同时中签率可能还更高。

## 五、暗盘（Grey Market）

### 5.1 定义

新股正式上市前一天（T-1日）16:15-18:30的场外交易市场，让中签者在上市前就能交易。

### 5.2 主要暗盘系统

| 系统 | 特点 |
|------|------|
| 辉立暗盘 | 老牌系统，流动性和价格可能优于富途 |
| 富途暗盘 | 散户为主，开盘即跑，价格往往偏低 |
| 耀才暗盘 | 另一主流暗盘系统 |

### 5.3 暗盘的作用

- **风向标**：暗盘与首日走势正相关比例达76%
- **提前锁定利润**：涨>20%可直接暗盘全出
- **止损出口**：破发可在暗盘立刻止损，不等首日

## 六、绿鞋机制（Greenshoe / 超额配售权）

### 6.1 定义

承销商在IPO中额外卖出不超过原定发行量15%股票的权利，用于上市后稳定股价。

### 6.2 运作原理

| 场景 | 操作 |
|------|------|
| 股价上涨 | 承销商行使绿鞋，从公司额外买入15%补仓 |
| 股价破发 | 承销商在二级市场买回股票支撑股价，绿鞋不行使 |

### 6.3 无绿鞋的含义

企业主动放弃超额配售权 → **传递强信心信号**，表达"不需要稳价兜底"的态度。无绿鞋新股首日上涨概率约77%（有绿鞋约60%）。

---

## 七、融资打新（孖展 / Margin）

### 7.1 基本概念

通过向券商借钱认购新股，放大申购金额。

### 7.2 关键参数

| 项目 | 说明 |
|------|------|
| 最高杠杆 | 10倍（2023年起） |
| 融资利率 | 年化约3% |
| 冻结天数 | 通常5-7天 |
| 利息公式 | 融资金额 × 年利率 × (冻结天数÷365) |

### 7.3 隐含风险

- **不中签也要付利息**：这是融资打新的隐性成本
- 融资100万港元，冻结5天 → 利息约411港元
- 预期暗盘涨幅<3%时，融资打新大概率亏损

## 八、其他重要概念

### 8.1 保荐人 / 稳价人（Sponsor / Stabilizing Manager）

| 角色 | 职责 |
|------|------|
| 保荐人（Sponsor） | 负责IPO全流程，对招股书内容质量负最终责任 |
| 稳价人（Stabilizing Manager） | 负责上市后执行绿鞋、稳定股价 |

保荐人历史破发率是重要参考指标。如中金、高盛、大摩等大行，历史破发率<20%时可信度高。

**保荐人黑名单**：历史破发率>50% → 一票否决。

### 8.2 老股转让（Sale of Old Shares）

老股东（创始团队、早期投资者等）在IPO时同步卖出自己持有的旧股。

| 比例 | 信号 |
|------|------|
| 无老股转让 | ⭐⭐⭐ 正面 |
| 老股<10% | ⭐⭐ 尚可 |
| 老股10%-25% | ⭐ 需警惕 |
| 老股>25% | ❌ 负面——老股东急于套现离场 |

### 8.3 18A/B类股

港交所允许未盈利的生物科技公司上市的特殊章节：

- **18A**：未盈利的生物科技公司
- **18B**：特殊目的收购公司（SPAC）相关

18A类高风险高回报：2025年上市的14家18A股赚钱效应极强，但需评估管线/临床数据质量。无实证数据的生物科技股破发率约58%。

### 8.4 认购倍数

公开认购总额 ÷ 公开发售额度。

| 倍数区间 | 解读 |
|---------|------|
| <15倍 | 冷门，回拨风险小但市场认可度低 |
| 15-100倍 | 中等，回拨压力增大 |
| 100-200倍 | "不冷不热"最危险区间 |
| >500倍 | 极端热度，抵消回拨压力，正收益确定性高 |

**口诀**："要么冷、要么热，最怕不冷不热。"

### 8.5 国配倍数

国际配售认购总额 ÷ 国际配售份额，反映机构参与热度。

| 倍数 | 信号 |
|------|------|
| >10倍 | 机构强烈看好 |
| 5-10倍 | 机构认可 |
| 1-5倍 | 中性偏弱 |
| <1倍 | 负面（但套路回拨时需另行判断） |

### 8.6 中签率

| 术语 | 说明 |
|------|------|
| 一手中签率 | 只申购1手的中签概率 |
| 稳中手数 | 至少中1手需要申购的手数 |

### 8.7 发行价区间与定价

港IPO在招股书里会公布一个**发售价区间**（如"每股 10-12 港元"），最终发行价需落在这个区间内。

| 概念 | 说明 |
|------|------|
| 发售价区间 | 招股书中公布的定价范围，下限到上限 |
| 上限定价 | 最终发行价 = 区间的最高值 |
| 下限定价 | 最终发行价 = 区间的最低值 |
| 中间定价 | 最终发行价在区间中间 |
| 上限定价 + 超上限 | 最终定价超出公开承诺的上限（需发布补充公告） |

**为什么有上限？**

发售价区间本质上是对市场的"价格承诺"——公司通过区间向投资者传达："我们认为合理的价格在这个范围内"。上限是这个承诺的天花板。

**超上限定价为什么危险？**

最终价格超出招股书承诺上限（如区间 10-12 元，结果定到 13.8 元）：

1. **变相加价**：坐地起价，公司和承销商在榨取最大利益
2. **不给二级市场留空间**：定价过高，上市后几乎没有上涨余地
3. **诚意问题**：招股书的价格区间形同虚设

**历史数据**：定价超上限 15% 的新股破发率约 **72%**，被列为 **一票否决项**。

## 九、买入卖出规则

### 9.1 交易时间线

```
招股期（3-5天）→ 定价日 → 暗盘日（T-1, 16:15-18:30）→ 上市首日（T日, 09:30起）
```

### 9.2 核心纪律

- **打新获利在暗盘或首日兑现，不要恋战**
- 暗盘涨>20%：全出，锁定利润
- 暗盘破发：立刻止损，不等首日
- 首日盘中从高点回撤>5%：立即市价卖出
- 任何时刻破发：无条件市价卖出

### 9.3 长期持有的风险

- 2024年首日上涨的股票中，**33.3%在年底跌破发行价**
- 多数新股"首日即巅峰"
